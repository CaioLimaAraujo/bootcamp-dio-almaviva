# Projetos de Código

Repositório de projetos desenvolvidos durante o **Bootcamp Backend Java & QA** — uma parceria entre a **DIO (Digital Innovation One)** e a **Almaviva**.

---

## Projetos

| # | Projeto | Módulo | Descrição |
|---|---------|--------|-----------|
| 01 | [Abstraindo um Bootcamp](modulo-03-orientacao-a-objetos/desafio-01-abstraindo-o-bootcamp) | Orientação a Objetos | Modelagem OOP de um bootcamp fictício aplicando os quatro pilares da POO com Collections. |
| 02 | [Sudoku em Java](modulo-05-sudoku-em-java) | POO e Estruturas de Dados | Jogo de Sudoku com interface gráfica Swing, separado em camadas model, service e view. |
| 03 | [Acervo API](modulo-06-acervo-api) | Design Patterns + Spring Framework | API REST de gerenciamento de obras intelectuais com Spring Boot e padrões GoF. |
| 04 | [Task Board Manager](modulo-07-board-manager) | Técnicas Avançadas e Persistência | Aplicação de console estilo Kanban com persistência em MySQL via JDBC. |
| 05 | [Testes de Software](modulo-08-testes-de-software) | QA | Simulação de QA manual funcional caixa-preta com JIRA, Confluence, MindMup e BDD/Gherkin. |
