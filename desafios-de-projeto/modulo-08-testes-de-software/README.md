# SwagLabs QA — Testes Manuais Funcionais

Projeto prático do módulo de QA do Bootcamp Java Backend & QA
(DIO + Almaviva), desenvolvido por Caio Lima Araújo.

## Sobre o Projeto

Simulação do dia a dia de um Analista QA em ambiente ágil (Scrum),
com foco em testes manuais funcionais do tipo caixa-preta,
aplicados ao sistema [SwagLabs Shopping](https://www.saucedemo.com/).

O projeto cobre as funcionalidades de Login e Carrinho de Compras,
passando por todas as etapas do processo de QA: levantamento de
requisitos, escrita de User Stories, planejamento, documentação
e casos de teste.

## Ferramentas Utilizadas

- **JIRA** — Gerenciamento de projeto e sprint (Atlassian)
- **Confluence** — Documentação colaborativa (Atlassian)
- **MindMup** — Mind map de cenários de teste
- **Gherkin / BDD** — Escrita de cenários de teste
- **Metodologia** — Scrum
