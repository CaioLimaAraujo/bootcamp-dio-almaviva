# Desafios de Código

Repositório de exercícios de lógica e sintaxe desenvolvidos durante o **Bootcamp Java Backend & QA**.

## Módulo 02 — Primeiros Passos na Sintaxe Java

Exercícios do módulo introdutório de Java, com foco em leitura de entrada, manipulação de strings e estruturas condicionais.

| # | Desafio | Descrição |
|---|---------|-----------|
| 01 | [Padronização de Nomes](modulo-02-fundamentos-java/desafio-01-padronizacao-nomes) | Lê o nome de um projeto e o retorna em letras maiúsculas seguido da quantidade de caracteres. |
| 02 | [Validação de Nomes](modulo-02-fundamentos-java/desafio-02-validacao-nomes) | Lê o nome de um projeto, remove espaços das bordas e o retorna em maiúsculas — ou `INVALIDO`, caso esteja vazio. |
