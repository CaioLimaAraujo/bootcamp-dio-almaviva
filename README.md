## Sobre o Bootcamp

**Desenvolvimento de Aplicações Backend em Java**, cobrindo desde os *Fundamentos de Java* até tópicos intermediários como *APIs REST*, *Persistência de Dados* & *Qualidade de Software*. Ao longo da formação, o aprendizado é consolidado e praticado através de *Desafios e Projetos de Código*.

A formação concluí com dois módulos inteiramente dedicados à área de **Garantia de Qualidade de Software**, onde o Primeiro Módulo fornece uma ampla visão da área, desde seus *Fundamentos Técnicos* até Projetos Práticos, com o Segundo Módulo aprofundando o tópico de *Testes de Software* em si.

Os seus Módulos foram os seguintes:

- Onboarding & Fundamentos de GitHub.
- Fundamentos de Java. (sintaxe & Estruturas de Controle)
- Orientação a Objetos com Java.
- Debug, Exceções, I/O & Gestão de Dependências em Java.
- Domínio de Collections e Streams API.
- APIs Rest com Java, Spring Boot & Persistência de Dados.
- Banco de Dados para Desenvolvimento Backend em Java.
- Fundamentos de Qualidade de Software.
- Testes de Software.
